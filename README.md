# ADFS Demo
## 專案結構
1. 測試資料庫 - Oracle 8i
2. 與資料庫連接的 Test API - ASP.NET MVC Web API(.NET Framework 4.5.2) + Dapper
3. 對外與UI互動的 Primary API - ASP.NET Core Web API(.NET Core 2.1)
4. UI - Angular 8 + Angular Material
